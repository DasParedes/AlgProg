dev-c++:

São apenas três arquivos:

conio.c     -> deve ser inserido na pasta C:\Dev-Cpp\include
conio.h     -> deve ser inserido na pasta C:\Dev-Cpp\include
libconio.a  -> deve ser inserido na pasta C:\Dev-Cpp\lib

Depois disso, para usar os comandos da biblioteca, como:
textcolor(), textbackground(), clrscr(), gotoxy(), etc...
basta inserir a nova biblioteca: include<conio.c>


codeblocks:
criei o projeto pelo codeblocks.
criei uma pasta dentro da pasta do projeto e coloquei a library dentro da pasta por questão de organização.
Project -> Build options... -> Linker Settings -> Link libraries, Add
e abrir (marcar) o arquivo libconio.a

Não sou especialista nisso, então não sei se funciona de outras formas, mas para esses arquivos da biblioteca conio.c funcionou para mim.
